# Hidden Failures and Robustness in Biomedical Foundation Models

## Project Description

This project investigates trustworthiness and hidden failure modes in vision-language models (VLMs), with a particular focus on biomedical and high-stakes decision-making settings. The project examines how models behave when visual evidence is missing, ambiguous, degraded, or misleading, and evaluates their ability to appropriately acknowledge uncertainty. Inspired by recent work on MIRAGE behavior and evidence failures, the project aims to identify situations in which models generate confident but unsupported responses despite insufficient visual information. The long-term goal is to develop a reproducible benchmark and evaluation framework for studying robustness, reliability, and trustworthiness in multimodal foundation models.

## Week 0 Goal

The goal of Week 0 is to establish a reproducible project infrastructure and validate the end-to-end evaluation pipeline. Specifically, we aim to verify that a vision-language model can process an image and question, generate a response, and save the result in the project's output format. If the selected VLM cannot be executed successfully, a fallback mock adapter will be used to generate schema-compliant outputs while documenting any encountered errors. Week 0 also serves as an onboarding phase for reproducing a small MIRAGE-style experiment before constructing larger benchmark components.
